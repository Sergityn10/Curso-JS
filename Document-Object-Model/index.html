Document Object Model

Existen distinto tipos de elementos dentro de documento HTML.

  1. documentElement: Hace referencia a cualquier elemento del documento HTML. Esta clase, todas tienen las propiedades head y body
    • nodeType: Devuelve el número del tipo de nodo que se trata.

Funciones para moverse por el DOM

Es muy importante para un desarrollador, tener los conocimientos para conocer las funciones específicas para desenvolverse por el DOM.

JS tiene unas funciones propias de cada elemento que nos proporciona la facilidad para alcanzar el objetivo de esta sección. Estas funciones son:

  1. childNodes: Devuelve la lista de objectos de los elemento hijos.
  2. children: Devuelve una lista de Node del elemento donde se está llamando.
  3. firstChildren y lastChildren: Devuelven el primer y último nodo hijo del elemento en el que se está llamando. Devuelve null, si no tiene nodos hijos.
  4. nextSibling y previousSibling: Devuelve el siguiente o anterior elemento que tiene el mismo padre, respectivamente. En caso de que sea el primer o último hijo, respectivamente, devolverán null

Objetos del DOM

El DOM está compuesto por una serie de objetos que representan los elementos del documento

The document and window objects are the objects whose interfaces you generally use most often in DOM programming

Reflected attributes

The Element.getAttribute() will return "" if the input is checked or null if it is not. The corresponding HTMLInputElement.checked property returns true or false for the checked state. Otherwise boolean reflected attributes are the same as any other reflected attributes.
